# ATM 管理系统

欢迎来到 ATM 管理系统项目！这是一个用 C 语言编写的模拟银行自动取款机（ATM）的程序，提供了用户和管理员操作界面。

## 功能特点

- 用户可以进行存款、取款、转账、查询余额、开户、销户和修改密码等操作。
- 管理员可以查看用户数据、进行用户数据分析、管理用户信息等。
- 系统提供了详细的使用帮助和操作指南。

## 快速开始

### 环境要求

- C 编译器（如 GCC）
- POSIX 兼容的操作系统（测试于 Linux）

### 安装步骤

1. 克隆仓库到本地机器
